# Quick Start Guide

## Get API Keys (2 minutes)

### 1. Google Gemini API
- Go to: https://aistudio.google.com/apikey
- Click "Create API Key"
- Copy the key

### 2. Firecrawl API
- Go to: https://www.firecrawl.dev
- Sign up (free account)
- Copy API key from dashboard

## Add Keys to v0

1. Click **"Vars"** in the left sidebar
2. Add two environment variables:
   - `GOOGLE_GEMINI_API_KEY` = [your gemini key]
   - `FIRECRAWL_API_KEY` = [your firecrawl key]
3. Done!

## Use the App

1. **Input** a blog URL (e.g., https://example.com/my-blog-post)
2. **Click** "Repurpose"
3. **Copy** any content you want with the copy button
4. **Paste** into LinkedIn, Twitter, or use YouTube description
